# Playlist WebApp
사용자가 자신만의 플레이리스트를 만들고 공유할 수 있는 웹 서비스

---

## Team
- 김해수: 프런트엔드 (UI 설계, HTML/CSS/JS)
- 김준성: 백엔드 및 데이터베이스 (Flask, SQLite3)

---

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Backend: Flask (Python)
- Database: SQLite3
